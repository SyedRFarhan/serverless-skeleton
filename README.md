# Setup

## Prerequisites

#### Accounts

- Github SSH Key
- Dada AWS account

#### Tools

- VSCode
- Nvm
- Node v14.4.0+
- Yarn

#### VSCode Extensions

- Eslint
- Prettier
- GitLens
- AWS Toolkit
- DotENV

## Installation

1. Clone this repo using SSH
2. At the root of the repo run 'yarn' to install dependencies for the monorepo
3. Create a `.env` file and add your aws credentials.
